Image Fetcher
Context-aware image search from Unsplash, Pexels, and Pixabay. Scans your project for context, generates smart queries, presents curated picks, and downloads with clean filenames. Zero config required.

Logo Generator

Presentation Chef
Apple Keynote-style HTML presentations with cinematic animations, glassmorphism, 5 themes, 13 slide types, speaker notes, and PDF export. Single self-contained .html file.

Video/Audio Processor
Transcribe recordings with local Whisper (no API key), extract visual frames with ffmpeg at 1 per minute, and synthesize everything into structured meeting notes with participants, decisions, and action items.
